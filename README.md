# Vendor Performance Analysis | Python | SQL | Power BI

An end-to-end data analytics project developed to evaluate vendor performance and procurement activities using **Python, SQL, and Power BI**. The project analyzes vendor ratings, purchasing trends, payment status, delivery performance, and category-wise spending through SQL queries and an interactive dashboard.

##  Project Highlights

- Analyzed **2,000 procurement transactions** across **100 vendors**.
- Performed data validation and exploration using **Python (Pandas)**.
- Developed **17 SQL queries** to analyze vendor performance and procurement trends.
- Built an interactive **Power BI dashboard** to visualize procurement insights and KPIs.
